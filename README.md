# YorTrade

YorTrade — это асинхронная биржевая игра-бот для групповых чатов.
Игроки торгуют акциями компаний по тикам: цены обновляются автоматически, есть новости/события, лидерборд и личные экраны.

## Где работает

- Telegram: группа + личные сообщения с ботом.
- VK: беседа + личные сообщения сообщества.
- Можно запускать сразу обе платформы (TG и VK) в одном процессе.

Важно: если токен платформы не задан, эта платформа просто не будет запущена.

## Как играть (быстрый сценарий)

1. Каждый игрок сначала пишет боту в личку (это регистрирует `dm_chat_id`).
2. Хост в групповом чате запускает лобби: `/start_game`.
3. Остальные присоединяются: `/join_game`.
4. Хост при необходимости меняет настройки через кнопки в лобби.
5. Хост запускает матч: `/begin_game`.
6. Игроки торгуют в личке через `/game` (или `/market`) и кнопки.
7. Завершение:
- хост: `/stop` (или `/end_game`),
- игрок: `/leave`.

## Команды

Префикс по умолчанию: `/` (настраивается через `PREFIX`).

| Команда | Где | Что делает |
|---|---|---|
| `/start_game` (`/start_gameYT`) | группа | Создать лобби игры |
| `/join_game` (`/join_gameYT`) | группа | Присоединиться к лобби |
| `/begin_game` | группа | Запустить игру (только хост) |
| `/stop` (`/end_game`) | группа | Завершить игру (только хост) |
| `/game` (`/market`) | личка | Открыть основной игровой экран |
| `/buy <asset_id> <amount>` | личка | Купить акции |
| `/sell <asset_id> <amount>` | личка | Продать акции |
| `/portfolio` | личка | Показать портфель |
| `/deals` | личка | Показать последние сделки |
| `/leaderboard` | личка | Показать лидерборд |
| `/leave` (`/выйти`) | обычно личка | Покинуть текущую игру |
| `/ping` | везде | Проверка, отвечает `PONG` |
| `/load_server_data` | везде | Загрузить компании/фразы (только admin id) |

## Запуск локально

### Требования

- Python `>=3.14`
- PostgreSQL
- Redis
- RabbitMQ

### 1) Установить зависимости

```bash
uv sync
```

### 2) Заполнить `.env`

Минимум:

```env
DATABASE_DSN=postgresql+asyncpg://postgres:postgres@localhost:5432/app
REDIS_DSN=redis://localhost:6379/0
RABBIT_DSN=amqp://guest:guest@localhost/

TG_TOKEN=
TG_BOT_USERNAME=
VK_TOKEN=
VK_GROUP_ID=0
ADMIN_LOGIN=admin
ADMIN_PASS=change_me

PREFIX=/
MIN_PLAYERS=1
ADMIN_TG_ID=0
```

Примечания:

- Для Telegram укажи `TG_TOKEN` (и желательно `TG_BOT_USERNAME`).
- Для VK укажи `VK_TOKEN` и `VK_GROUP_ID`.
- Для админки обязательно укажи `ADMIN_LOGIN` и `ADMIN_PASS`.
- Для запуска сразу на двух платформах заполни оба набора.

### 3) Применить миграции

```bash
uv run alembic upgrade head
```

### 4) Запустить приложение

```bash
uv run python main.py
```

Сервис стартует на `0.0.0.0:8080`.

### 5) (Первый запуск) загрузить игровые данные

После старта выполни команду `/load_server_data` из админ-аккаунта (`ADMIN_TG_ID`).
Без этого в пустой БД матч не соберется (не будет компаний для рынка).
